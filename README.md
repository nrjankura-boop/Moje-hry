# Moje-hry
NoroDB je pokus o databazovu aplikaciu podobnu MySQL alebo inej SQL databaze. V prilozenej dokumentacii je vysvetlene co to dokaze.
Je to uplny zaklad. Vyvijam novsiu verziu tejto aplikacie, v ktorej je moznost mazat data z tabulky, aj pomocou podmienky WHERE. A ine 
vymozenosti...
Zatial je dotaz uplne na konci suboru NoroDB_ver1.0.1.py ako retazec, po spusteni suboru sa vykonaju dotazy v tomto retazci. Pre select
dotaz vypise udaje z tabulky na obrazovku. Planujem vyuzit tkinter gui na vylepsenie zobrazovania vysledkov dotazov, resp. udajov z tabulky.
